This design takes inspiration from:
- **Outline**: Clean sidebar, modern typography
- **Notion**: Card-based layouts
- **Linear**: Color scheme and polish
- **GitHub**: Professional, developer-friendly aesthetic

The result is a professional, modern wiki interface that feels familiar yet distinctive.
